# TaskFlow

> מערכת אישית לניהול פרויקטים, משימות ולקוחות

## סטאק

- **Backend:** FastAPI (Python 3.11)
- **Database:** MongoDB Atlas
- **Frontend:** HTML + Tailwind CSS + Alpine.js
- **Hosting:** Render
- **Auth:** סיסמה אחת + session cookie

## API Endpoints

### לקוחות
- `GET /api/clients` - רשימת לקוחות עם סטטיסטיקות
- `GET /api/clients/select-options` - רשימה מצומצמת לטפסים
- `POST /api/clients` - יצירת לקוח חדש
- `GET /api/clients/{id}` - לקוח לפי ID
- `PUT /api/clients/{id}` - עדכון לקוח
- `DELETE /api/clients/{id}` - מחיקת לקוח (פרויקטים מתנתקים)

### פרויקטים (חדש)
- `GET /api/projects` - רשימת פרויקטים (ברירת מחדל: פעילים + בהמתנה)
- `GET /api/projects?include_inactive=true` - כולל הושלמו ובארכיון
- `POST /api/projects` - יצירת פרויקט חדש
- `GET /api/projects/{id}` - פרטי פרויקט עם סטטיסטיקות
- `PUT /api/projects/{id}` - עדכון פרויקט
- `DELETE /api/projects/{id}` - מחיקת פרויקט (כולל המשימות שלו)
- `GET /api/projects/{id}/tasks` - רשימת משימות הפרויקט

### עמודים
- `/` - דשבורד
- `/clients` - ניהול לקוחות
- `/projects` - רשימת פרויקטים
- `/projects/{id}` - פרטי פרויקט
- `/login` - התחברות

## הוראות שדרוג מגרסה קודמת

ב-Render, פשוט דחוף את הקוד החדש לריפו ו-Render יעשה rebuild אוטומטי.
לא צריך לעדכן env variables - אין שינויים בהגדרות.

## משתני סביבה נדרשים
- `MONGODB_URL` - connection string מ-Atlas
- `APP_PASSWORD` - הסיסמה שלך
- `SESSION_SECRET` - יוצר אוטומטית ע"י Render
