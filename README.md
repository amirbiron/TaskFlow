# TaskFlow

> מערכת אישית לניהול פרויקטים, משימות ולקוחות

## סטאק

- **Backend:** FastAPI (Python 3.11)
- **Database:** MongoDB Atlas
- **Frontend:** HTML + Tailwind CSS + Alpine.js
- **Hosting:** Render
- **Auth:** סיסמה אחת + session cookie

## מצב נוכחי - שלב 2 ✅

## מה השתנה מגרסה 0.1.0

1. תוקן באג שגרם ל-500 בבקשות HEAD לדף הבית
2. נוסף שדה `color` לכל לקוח
3. נוסף router לניהול לקוחות (CRUD מלא)
4. נוספה תבנית clients.html עם UI מלא
5. עודכנה תבנית הבסיס עם ניווט רספונסיבי

## הוראות שדרוג מגרסה קודמת

ב-Render, פשוט דחוף את הקוד החדש לריפו ו-Render יעשה rebuild אוטומטי.
לא צריך לעדכן env variables - אין שינויים בהגדרות.

## הוראות התקנה ראשונית

ראה את הוראות שלב 1 - לא השתנה.

### משתני סביבה נדרשים
- `MONGODB_URL` - connection string מ-Atlas
- `APP_PASSWORD` - הסיסמה שלך
- `SESSION_SECRET` - יוצר אוטומטית ע"י Render

## API Endpoints

### לקוחות
- `GET /api/clients` - רשימת לקוחות עם סטטיסטיקות
- `POST /api/clients` - יצירת לקוח חדש
- `GET /api/clients/{id}` - לקוח לפי ID
- `PUT /api/clients/{id}` - עדכון לקוח
- `DELETE /api/clients/{id}` - מחיקת לקוח (פרויקטים מתנתקים)

### עמודים
- `/` - דשבורד
- `/clients` - ניהול לקוחות
- `/login` - התחברות
